Heatmap zoom & interactivity

New feature on the freshly-merged viz-web redesign. Let's scope it.

What the heatmap does today

Current LandscapeMap — shipped in the redesign

Visualization: D3 force-simulated bubble graph, 180 ticks, rendered as inline SVG with a fixed viewBox.

Interactions today: hover a bubble → tooltip + dim non-related; click a bubble → navigate to /node/:kind/:slug; table ↔ map hover sync via parent state.

Not interactive: no pan, no zoom, no node drag, no keyboard controls, no way to see details without leaving the page.

Scale: currently bounded to whatever fits the fixed 720 × 520 viewBox. A snapshot with 50+ nodes gets crowded fast.
